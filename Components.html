Instrument Driver Components

GUI

The GUI presents instrument information to the operator.

GUI Driver

The GUI driver is code that is part of the GUI. It's job is to present data from the instrument driver on the GUI.

Configs

Configuration files determine how the driver interacts with the instrument. They determine the communication transport, the parsers, the scripts which implement the standard features of the driver.

GUI Data

The GUI Data is a standardized dataset that is produced by the driver and consumed by the GUI.
